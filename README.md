## 使用前提

### 1：于矿机所在的内网准备一台主机（提供矿机连接）
  windows & linux 都兼容，建议用linux（windows不好做到无人值守）
### 2：服务器部署代理，（提供矿池连接，并给内网机器发送矿池数据）
  一定用linux，无人值守
